# Stay Smitten Sign Up Page
This is the the front-end for the Stay Smitten Sign Up Page.

Visit [Stay Smitten](https://staysmitten.com/) for more details.

## Features
- Sign Up For New Stay Smitten Members
- Responsive on mobile devices up to iPhone 5 size
- Protected route for viewing data
  - Login Page
  - React Prime Table
